# DwellTend

DwellTend is a warm, practical home-care dashboard for shared households. It starts with ready-made cleaning, maintenance, calendar, and inventory routines so a household can launch without building its system from scratch.

![DwellTend logo](public/brand/logo-horizontal.png)

## What is included

- Guided onboarding for household size, home type, and pets.
- Preloaded task templates across cleaning, maintenance, inventory, seasonal, and admin work.
- Dashboard views for open tasks, overdue work, weekly planning, supplies, and household load.
- Brand assets, favicon, app icon, and visible in-app logo treatment.
